============
Proya数据库
============

Proya项目是一个业务复杂，多类型数据库的海量数据维护、管理和分析项目，其中涉及到CRM、SRM、企业OA等关联数据库。这个项目主要是根据相关数据库整理的字段管理表，方便数据库设计、开发以及后续报表实现。
